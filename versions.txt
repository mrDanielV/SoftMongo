Версия 1.0.0, 25.11.2022
	Первая версия системы

Версия 1.1.0 15.12.2022
	- Фильтрация в списке коллекций
	- Интерактивный объект записи БД:
		Подсказка "Дата" для целочисленных узлов в LOCAL таймзоне
		Добавление в меню действие с узлом пункта "Добавить к запросу". Возможность добавить выбранный узел (путь:значение) к текущему запросу
	- Экспорт и Импорт коллекций/записей в БД, в т.ч. экспорт результата текущего запроса
	- Возможность менять мышкой ширину блока списка коллекций

Версия 1.2.0 24.01.2023
	- Исправление ошибок
	- Оптимизация и рефакторинг PHP-классов, обеспечивающих выполнение операций UI
	- Импорт файлов-дампов БД до 256М (при условии наличия гибкого восприятия WEB-сервером изменений констант upload_max_size, upload_max_filesize, post_max_size)

Версия 1.2.1 21.02.2023
	- Возможность сворачивать/разворачивать все блоки выбранного уровня в интерактивном объекте записи коллекции
	- Возможность задавать LIMIT (от, до) при экспорте коллекции
	- Подсказки для полей форм "Сортировка" и "Добавление индекса" с вариантами путей в объектах коллекции 
	- Сохранение в COOKIE текущего сервера и БД, автоматическое их применение как текущих при открытии программы
	- Вывод имени сервера над именем текущей БД в модуле БД

Версия 1.2.2 27.02.2023
	- Возможность создания "уникальных" индексов (запрет на создание дубликатов записей по указанному ключу)
	- Регистронезависимый поиск по имени коллекции
	- Исправление ошибки фиксации текущей страницы при изменени запроса

Версия 1.2.3 02.03.2023
	- Устранение критической ошибки изменения записи при наличии настройки "Поля"
	- Возможность копировать объект записи в виде JSON в буфер обмена
	- Использование текущей настройки "Поля" при "Обновлении" записи

Версия 1.3.0 30.05.2023
	- Раздел "Агрегация" в коллекциях БД с соответствующим функционалом db.collection.aggregate()
	- Аналитика (Explain) запросов типа FIND - db.collection.explain().find()
	- Незначительные изменения по дизайну

Версия 1.3.1 16.06.2023
	- Применение к SoftMongo новой версии фреймворка Резолют
	- Незначительные изменения в дизайне
	- Исправление ошибки операции "Переименовать поле" в блоке действий с узлом объекта записи БД

Версия 1.3.2 10.07.2023
	- Устранения ошибки загрузки плана агрегации из JSON
	- Добавление возможности сброса текущей настройки Сортировки и Полей запроса
	- Выделение жирным шрифтом текущей коллекции в списке коллекций текущей БД
	- Возможность фильтрации в списке типов этапов агрегации при добавлении этапа 
	- Изменение дизайна основного меню действий по коллекции
	- Включение пункта Экспорт в основное меню действий по коллекции
	- Получение и вывод версии сервера MongoDB в разделах "Сервера" и "Базы данных сервера"

Версия 1.3.3 03.10.2023
	- Исправление ошибок
	- Обновление количества записей коллекции в списке коллекций при любом запросе к коллекции

Версия 1.3.4 07.11.2023
	- Исправление ошибки работы модуля агрегации

Версия 1.4.0 25.12.2023
	- Полная замена серверного обеспечения работы с MongoDB, отказ от resolute_mongodb.php, переход на собственные классы-драйвера SoftMongo
	- Изменение формата конфигурации подключения к БД
	- Поддержка PHP 5.x и Драйвера \MongoClient
	- Отладка работы SoftMongo на PHP 8.x
	- Общая оптимизация скорости работы пользовательского интерфейса

Версия 1.5.0 09.07.2024

	- Универсализация определения корневой директории системы и подключения классов

Версия 1.5.1 17.09.2024
